# chan models

何となーく作った3Dモデルを表示したり保存したりできるアプリだよ

## 概要

- 組み込まれた3Dモデルを選んで、その場で3D表示できるよ
- **STL** または **色付き3MF** で保存できるよ

## 導入方法

[Releases](https://github.com/misetteichan/chan-models/releases) から
お使いのOSのzipをダウンロードして展開してください。

### macOS (Apple Silicon)

1. `chan-models-macos.zip` を展開し、`chan-models.app` を好きな場所へ
   (アプリケーションフォルダなど)
1. 初回は「悪質なソフトウェアがないか確認できない」等の警告が出て開けません
   (このアプリは署名・公証を行っていないため)。次の手順で起動してください
   1. 一度ダブルクリックし、出てきた警告を閉じる
   1. **システム設定 → プライバシーとセキュリティ** を開き、下の方にある
      「"chan-models" はブロックされました」の **「このまま開く」** をクリック
   1. 確認ダイアログで「開く」を選ぶ(パスワードまたはTouch IDを求められます)
1. 2回目以降は普通にダブルクリックで開けます

> macOS 14 (Sonoma) 以前では、**右クリック(Controlクリック)→「開く」** でも
> 起動できます(15以降ではこの方法は使えません)

### Windows

1. `chan-models-windows.zip` を展開し、フォルダごと好きな場所へ
1. `chan-models` フォルダ内の `chan-models.exe` を実行
1. 初回に SmartScreen の警告が出た場合は
   **「詳細情報」→「実行」** で起動してください(未署名アプリのため)

> うまく起動しない場合は、zip を**展開する前に** 右クリック → プロパティ →
> 「許可する」(ブロックの解除)にチェックを入れてから展開し直してみてください

## 使い方

- **モデルを選ぶ** — 左上のリストから切り替え
- **3Dビューで確認** — ドラッグで回転、ホイールでズーム、
  右ドラッグ(または2本指ドラッグ)で移動。「全体表示」でカメラが戻ります
- **パラメータを調整** — 「パラメータを調整する」にチェックを入れると
  スライダーが現れ、動かすと形が即座に更新されます。
  「初期値に戻す」でいつでもやり直せます
- **パーツの表示切替** — 複数パーツのモデルは、チェックで個別に表示/非表示。
  非表示のパーツは保存されません
- **保存** — 「STL 書き出し」または「3MF 書き出し」
  - **3MF**: パーツの色・名前を保ったまま1ファイルに保存
  - **STL**: パーツが1つなら1ファイル。複数のときは保存先フォルダを選ぶと
    `モデル名_パーツ名.stl` としてパーツごとに保存されます

## 注意事項

- 寸法の単位はすべて **mm** です
- macOS は **Apple Silicon (M1以降) 専用**です。Intel Macでは動作しません
- Windows 版の `_internal` フォルダはアプリ本体の一部です。
  削除したり exe だけ移動したりすると起動しなくなります
- 3MF の読み込みは **Bambu Studio で動作確認済み**です。
  他のスライサー(PrusaSlicer / OrcaSlicer 等)でも読めるはずですが未確認です
- 出力した3Dモデルの印刷可否・強度・寸法精度は、プリンタやスライサー設定に
  依存します。印刷前にスライサー上でサイズをご確認ください
